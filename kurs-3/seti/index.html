Лабораторная работа №4

Определение MAC и работа с ARP:

  • поиск MAC-адреса для своего активного сетевого интерфейса (CLI | GUI);
  • выполнить любой запрос к узлу той же локальной сети, где находишься. (команда ping);
  • открыть таблицу ARP и выяснить физический адрес узла, к которому делали ping.

Лабораторная работа №5

Мониторинг сети. С помощью программы Wireshark исследовать сетевую активность на своем компьютере, список хостов и протоколов, которые участвуют в информационном обмене. Попробовать подключиться к любому сайту по протоколу HTTP и убедиться в том, что ваши данные передаются в открытом виде.

Лабораторная работа №6

Исследование транспортного уровня. Провести исследование открытых соединений на компьютере с помощью утилиты Netstat:

  • просканировать порты любого хоста;
  • для каждого из обнаруженных портов указать какой сервис вероятнее всего запущен на исследуемых хостах.
