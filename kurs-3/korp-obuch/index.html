Основы корпоративного электронного обучения

Здесь представлены работы по дисциплине "Основы корпоративного электронного обучения" за 3 курс.

Практические работы

Практическая работа №1
  • Описать принципы корпоративного обучения;
  • Написать какое место в системе управления персоналом современной организации занимает подсистема корпоративного обучения;
  • Раскрыть содержание основных требований к организации корпоративного обучения;
  • Определить персонифицированную ответственность за решение задач обучения: стратегических, исследовательских, методических и организационных.
  • Определить основные особенности каждого из этапов в цикле системного обучения.
  • Раскрыть содержание основных этапов в эволюции корпоративного обучения.
