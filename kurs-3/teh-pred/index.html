Тема 4. Автоматизация представления данных средствами электронных таблиц

Работа №1

Задание 1. Заполнить пустые закрашенные ячейки в диапазоне AD4:BE5, придумав номера рабочих учебных планов из 5 цифр. Задание 2. Заполнить пустые закрашенные ячейки в диапазоне AD11:BE12, придумав количество студентов для конкретного года поступления (от 10 до 60). Задание 3. Заполнить столбик "W", используя значения из диапазона AD11:BE12. Задание 4. Заполнить столбики "X и "Y", используя значения из столбца "W". Задание 5. На отдельном листе отобразить значения по следующим требованиям:

  • Значения в диапазонах АD4:ВE5, АD11:ВE12, BG3:BH15 берутся с листа «Данные».
  • Значения столбцов «А – L» и «Q – AA» берутся с листа «Данные».
  • Значения с столбце М вычисляются по формуле 10% от (количество лекций + количество практик + количество лабораторных).
  • Значения с столбце N вычисляются по формуле 2 + (0.33 часа на 1 студента).
  • Значения с столбцов O и P вычисляются по формуле (0.25 часа на 1 студента).
